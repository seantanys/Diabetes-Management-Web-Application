<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/seantanys/Diabetes-Management-Web-Application
```

Go to the project directory

```bash
  cd Diabetes-Management-Web-Application
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
  or
  nodemon app.js
```
